Important Concepts

How the Web Works

You use your computers browser to make an HTTP request via the internet. The internet makes the request to servers which respond with files (such as HTML Documents) and sends them back to the browser to be displayed on your screen.

HTML

HTML stands for Hyper Text Markup Link and is comprised of four main components:

1. Text Content: What you see
2. Markup: What it looks like
3. References to Other Documents: e.g. images and videos
4. Links to Other Pages

HTML is the basis for the majority of webpages.

Tags and Elements

Tags and elements are part of HTML markup.

Tags alter the way certain content is displayed.

The term element refers to the tag and the content within the tag.

Why Computers Are Stupid

Computers are literal and non-intuitive. They rely solely on human instruction in order to function. If a human error is made the computer will not know how to properly behave until the error is rectified by a human.

Inline vs. Block Elements

HTML elements are considered either inline or block. Block elements form an invisible box of width and height around the content.
